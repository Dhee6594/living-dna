# Getting Started

Zero-to-genome in a few minutes.

```bash
pip install -e .            # wires the `dna` CLI (engine is stdlib-only)
dna ingest /path/to/repo    # replay history → bitemporal graph → DNA profiles
dna profile <service>       # read a materialized profile
dna serve                   # JSON API + Genome Browser at http://127.0.0.1:8000
```
